```bash
Features of ChowdhuryVai Code Analyzer:
Professional Hacker-Themed Interface:

Dark background with green and blue accent colors

Terminal-style font for authentic hacker feel

Clean, organized layout

Advanced File Analysis:

Scans PHP, HTML, CSS, and JavaScript files

Detects potential security issues

Counts lines, functions, and variables

Identifies suspicious code patterns

Complete Folder Processing:

Recursively scans selected folders

Copies all analyzed files to output directory

Preserves folder structure

Detailed Reporting:

Generates comprehensive analysis report

Highlights potential security risks

Provides file statistics and metrics

Branding Elements:

ChowdhuryVai branding throughout the interface

Contact information in footer

Professional styling

User-Friendly Features:

Progress bar for long operations

Status updates during analysis

Clear results display with color coding

Ability to clear results and start over

How to Use:
Run the Python script

Select the folder containing your code files

Choose which file types to analyze

Specify an output folder for the results

Click "Start Analysis"

View the results in the interface and in the generated report
```
